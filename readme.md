# 🧠 Emotion Sentiment Analyzer

A machine learning web application that detects human emotions from text using Scikit-Learn and Streamlit.

## 🚀 Features
* **Real-time Analysis:** Enter any text and get instant emotion predictions.
* **Multi-class Classification:** Detects 6 distinct emotions (Joy, Sadness, Anger, Fear, Love, Surprise).
* **Probability Scores:** Shows how confident the model is about each prediction.
* **Clean UI:** Built with Streamlit for a responsive, modern experience.

## 🛠️ Tech Stack
* **Language:** Python
* **ML Library:** Scikit-Learn (Logistic Regression)
* **Frontend:** Streamlit
* **Data Processing:** Joblib, Pandas
